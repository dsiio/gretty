# testTomcatContextConfig

Example of configuring webapp context via tomcat-specific configuration file "context.xml".

## How to run

```bash
cd examples/testTomcatContextConfig
gradle appRun
```

## How to test

```bash
cd examples/testTomcatContextConfig
gradle integrationTest
```

## How to build a product


```bash
cd examples/testTomcatContextConfig
gradle buildProduct
```
