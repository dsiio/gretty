# helloJersey

Minimalistic Jersey web-service running on Gretty.

## How to run

```bash
cd examples/helloJersey
gradle appRun
```

open in browser: http://localhost:8080/helloJersey

## How to test

```bash
cd examples/helloJersey
gradle integrationTest
```

## How to build a product


```bash
cd examples/helloJersey
gradle buildProduct
```
