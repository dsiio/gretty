# testJettyServerConfig

Example of configuring jetty server via jetty-specific configuration file "jetty.xml".

## How to run

```bash
cd examples/testJettyServerConfig
gradle appRun
```

## How to test

```bash
cd examples/testJettyServerConfig
gradle integrationTest
```

## How to build a product


```bash
cd examples/testJettyServerConfig
gradle buildProduct
```
