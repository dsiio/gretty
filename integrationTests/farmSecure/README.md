# secureFarm

Example of webapp farm, secured with HTTPS protocol and security realms.

## How to run

```bash
cd examples/secureFarm/MyWebApp
gradle farmRun
```

## How to test

```bash
cd examples/secureFarm/MyWebApp
gradle farmIntegrationTest
```

## How to build a product


```bash
cd examples/secureFarm/MyWebApp
gradle buildProduct
```
